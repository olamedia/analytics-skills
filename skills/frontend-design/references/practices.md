## How to use

**Audit mode:** Read each relevant category guide. The checklist table at the top of each file is the fast scan. The detailed guidance below explains edge cases and common mistakes.

**Active guidance mode:** Internalize the checklists for categories relevant to the current task. Read detailed guidance only when a decision is non-obvious.
